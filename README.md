# Tempest Español

Este proyecto es una versión traducida al Español (México) del launcher de Tempest, con el objetivo de facilitar su uso a personas que tienen dificultades con el inglés.

Ninguna de las funcionalidades del launcher fueron modificadas.

Es importante aclarar que este launcher **aún se encuentra en desarrollo y no está finalizado**. Se utilizó como base el launcher original de Kana, el cual no es open-source.

Si hay algún error de traducción el cual se me pasó por arriba o hay algo que no está traducido, puedes unirte al [servidor de Discord](https://discord.gg/FB9faSwHuB) y mencionarme allí.

# Modo de uso

Para usar el launcher, puedes hacerlo de dos formas:

1. Puedes ir directamente hacia la página de [releases](https://github.com/tx-ro/Tempest-Spanish/releases) y [descargar la última versión disponible](https://github.com/tx-ro/Tempest-Spanish/releases/latest).

2. Puedes compilar el launcher desde el código. Aquí tienes una [guía](https://github.com/tx-ro/Tempest-Spanish/blob/master/.github/CONTRIBUTING.md). Cabe de mencionar que la guía está en inglés, así que si no entiendes que hay que hacer te recomiendo simplemente descargar el launcher de la otra manera.
